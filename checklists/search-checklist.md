# Чек-лист: Поиск фильма

## 🔹 Поле поиска
- Отображение строки поиска
- Наличие placeholder
- Возможность ввода текста
- Ограничение длины ввода

## 🔹 Ввод данных
- Пустой ввод
- Валидный ввод
- Ввод с опечаткой
- Несуществующее значение
- Спецсимволы
- Числа
- Разные языки (RU/EN)

## 🔹 Результаты поиска
- Отображаются релевантные результаты
- Корректное сообщение при отсутствии результатов
- Частичное совпадение
- Обработка опечаток (если предусмотрено)

## 🔹 UI / UX
- Подсветка совпадений
- Отображение постеров и названий
- Кликабельность результатов

## 🔹 Поведение системы
- Поиск по Enter
- Поиск по кнопке
- Скорость ответа
- Обработка ошибок

## 🔹 Граничные случаи
- Очень длинный запрос
- Быстрый ввод (debounce)
- Очистка строки поиска
- Повторный поиск
