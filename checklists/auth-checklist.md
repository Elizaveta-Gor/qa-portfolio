# Чек-лист: Авторизация

## 🔹 Форма входа
- Поле для ввода email отображается корректно
- Поле для ввода пароля отображается корректно
- Кнопка «Войти» присутствует и доступна для нажатия

## 🔹 Ввод данных
- Вход с валидным email и паролем
- Вход с неверным паролем
- Вход с несуществующим пользователем
- Вход с пустыми полями
- Ввод некорректного email (неверный формат)

## 🔹 Валидация
- Отображаются сообщения об ошибках
- Корректная валидация формата email
- Проверка обязательности заполнения полей

## 🔹 Поведение системы
- Успешная авторизация пользователя
- Блокировка после нескольких неудачных попыток (если предусмотрено)
- Сохранение пользовательской сессии
- Корректная работа выхода из аккаунта (логаут)

## 🔹 UI / UX
- Сообщения об ошибках понятны и отображаются корректно
- Поля подсвечиваются при ошибке
- Кнопка «Войти» меняет состояние (активна/неактивна)

## 🔹 Граничные случаи
- Ввод очень длинного email и пароля
- Использование спецсимволов
- Копирование и вставка данных в поля
