# Тест-кейсы: Поиск фильма

## Тест-кейс: Поиск с разными типами входных данных

### Предусловие:
Пользователь авторизован

### Тестовые данные:

| Тип ввода        | Значение        | Ожидаемый результат                  |
|------------------|-----------------|-------------------------------------|
| Валидный         | Гарри Поттер    | Фильм найден                        |
| Опечатка         | Гарри Потерр    | Рекомендации или релевантные фильмы |
| Несуществующий   | прмл4367        | Ничего не найдено                   |
| Спецсимволы      | @#$%            | Ошибка или пустой результат         |
| Пустой           | (пусто)         | Валидация или рекомендации          |

### Шаги:
1. Открыть страницу поиска
2. Ввести значение из таблицы тестовых данных
3. Нажать кнопку «Найти»

### Ожидаемый результат:
Результат соответствует ожидаемому результату из таблицы
